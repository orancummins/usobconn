Monarch Connection Tracker Financial Institution Status Monitor

How Scoring Works

This tool scrapes Monarch's Connection Status page and computes quality scores for each data provider (Plaid, Finicity, MX).

What Monarch Shows

For every financial institution, Monarch displays three quality metrics using a 4-block rating bar:

  • Success Rate — How reliably the connection works when users try to link their account.
  • Longevity — How long connections stay active and healthy over time.
  • Update Frequency — How often the connection pulls fresh transaction and balance data.

How We Read the Bars

Each metric is shown as 4 small colored blocks. We count how many blocks are filled (non-gray) and convert to a percentage:

  • 0 blocks filled = Not rated (excluded from averages)
  • 1 block = 25%
  • 2 blocks = 50%
  • 3 blocks = 75%
  • 4 blocks = 100%

Institutions with zero filled blocks are treated as unrated for that metric — they are not counted as 0%, they are simply excluded from the average. This avoids penalizing providers that serve many niche institutions that Monarch hasn't rated yet.

Provider Weighted Score

For each provider, we average each metric across all rated institutions, then combine them with these weights:

Weighted Score = (Success × 40%) + (Longevity × 30%) + (Updates × 30%)

Success is weighted highest because it directly reflects whether users can connect their accounts. The tooltips on the chart show exactly how many institutions were rated for each metric.

Scrape History

Each scrape is timestamped and stored. The history panel at the bottom compares runs over time, with delta arrows showing how each metric changed from the previous scrape. This lets you track whether providers are improving or degrading.

Data is scraped directly from Monarch's public page. Scores may vary between runs as Monarch updates their ratings. The scraper scrolls through the full page to capture all institutions accurately.
